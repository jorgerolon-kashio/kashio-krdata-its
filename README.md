# KRDATA Portal

Portal frontend para operar catalogos KRDATA (Geography, Financial, Business y Process), con API playground sobre el contrato de `docs/krdata.yaml`.

## Ejecucion local

1. Instalar dependencias: `npm install`
2. Iniciar servidor de desarrollo: `npm start`
3. Abrir en el navegador: `http://localhost:4200`

Por defecto consume `http://localhost:3001/api/v1` cuando no se inyecta `IT-KRDATA-API_BASE_URL`.
