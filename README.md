# Kbatch IT Service Portal - Dashboard Admin

Esta es una aplicación de administración de pagos moderna y limpia, inspirada en la interfaz de 'Kashio', construida con Angular 19.

## Características

- **Layout de Dos Columnas**: Sidebar izquierdo oscuro con navegación intuitiva.
- **Header Minimalista**: Con buscador, notificaciones y perfil de usuario con Breadcrumbs dinámicos.
- **KPI Cards**: Resumen ejecutivo con iconos circulares y tipografía destacada.
- **Gráfico de Tendencias**: Gráfico de área spline suave utilizando ApexCharts.
- **Tabla de Transacciones**: Diseño "flat" con etiquetas de estado (badges) redondeadas.

## Tecnologías Utilizadas

- **Angular 19**: Framework core con componentes standalone.
- **Lucide Icons**: Iconografía de línea fina y elegante.
- **ApexCharts**: Visualización de datos moderna.
- **Vanilla CSS**: Sistema de diseño basado en variables CSS para máxima personalización.
- **Inter Font**: Tipografía moderna de Google Fonts.

## Estilo Visual

- **Paleta de Colores**: Azul corporativo vibrante (#0052FF), fondo gris ultra claro (#F4F7FE) y blanco puro.
- **Bordes**: Esquinas muy redondeadas (12px - 20px).
- **Sombras**: Soft shadows que proporcionan profundidad sin peso visual.

## Cómo ejecutar

1. Instalar dependencias: `npm install`
2. Iniciar servidor de desarrollo: `npm start`
3. Abrir en el navegador: `http://localhost:4200`
